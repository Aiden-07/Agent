概览

欢迎使用智能体平台 API。通过 API，您可以将智能体集成到任何应用程序中。 所有 API 请求都应使用 https://api.vagent.ai/v1 作为基础 URL。

鉴权

使用 Bearer Token 进行身份验证。请在请求头中包含您的 API Key。 

Authorization: Bearer YOUR_API_KEY

发送对话 (Chat Completions)

向智能体发送消息并获取回复。支持流式传输 (SSE)。

POST /chat/completions

Request Body

{
  "agent_id": "ag_839201",
  "messages": [
    {"role": "user", "content": "Hello!"}
  ],
  "stream": true
}
